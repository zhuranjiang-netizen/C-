## **解题代码（C++）**

```cpp
class Solution {
public:
    string findDifferentBinaryString(vector<string>& nums) {

        int n = nums.size();
        string res="";

        for(int i=0;i<n;i++)
        {
            if(nums[i][i]=='0')
                res+='1';
            else
                res+='0';
        }

        return res;
    }
};
解题思路：
康托对角线，取nums[i][i]的数字取反放入res中，最后得到的就是不在字符串中的。
