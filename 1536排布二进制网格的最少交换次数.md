## **解题代码（C++）

```cpp
class Solution {
public:
    int minSwaps(vector<vector<int>>& grid) {
        class Solution {
public:
    int minSwaps(vector<vector<int>>& grid) {
        // 预处理每一行的尾零个数
        int n = grid.size();
        vector<int> tail_zeros(n, n);
        for (int i = 0; i < n; i++) {
            for (int j = n - 1; j >= 0; j--) {
                if (grid[i][j]) {
                    tail_zeros[i] = n - 1 - j;
                    break;
                }
            }
        }

        int ans = 0;
        for (int i = 0; i < n - 1; i++) {
            int j = i;
            while (j < n && tail_zeros[j] < n - 1 - i) {
                j++;
            }
            if (j == n) {
                return -1;
            }
            ans += j - i;
            // 从 j 换到 i，原来 [i, j-1] 中的数据全体右移一位
            copy_backward(tail_zeros.begin() + i, tail_zeros.begin() + j, tail_zeros.begin() + j + 1);
        }
        return ans;
    }
};

解题思路：
这可以贪心地做：从左到右，找第一个满足 tailZeros[i]≥n−1 的数，交换到第一个位置；然后找第一个满足 tailZeros[i]≥n−2 的数，交换到第二个位置；依此类推。
为什么要选最近的？假设下标 p 和 q 都可以换到下标 0。由于越靠后，tailZeros[i]≥n−1−i 这个约束越宽松，所以能换到下标 0，也能换到后面所有的下标，所以下标 p 和 q
能发挥的作用是一样的。既然一样，谁离下标 0 更近就选谁换过去。
由于 tailZeros[n−1]≥0 一定成立，所以外层循环只需循环到 i=n−2。
