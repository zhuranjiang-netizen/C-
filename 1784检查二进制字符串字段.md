## **解题代码（C++）**

```cpp
class Solution {
public:
    bool checkOnesSegment(string s) {
     int cnt=0;
     int count=0;
     for(int i=0;i<s.size();i++)
     {
        if(s[i]=='1')
        cnt++;
        else
        {
            if(cnt!=0)
            {
                count++;
                cnt=0;
            }    
        }
        if(cnt!=0&&i==s.size()-1)
        count++;
     }
     if(count>1)
     return false;
     else
     return true;
    }
};
解题思路：
用cnt去记录一下当前连续1的长度，如果不等于0，就证明是一条，用count记录，看最后结果是否大于1.
