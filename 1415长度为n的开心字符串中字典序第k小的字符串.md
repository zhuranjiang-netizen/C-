## **解题代码（C++）**

```cpp

class Solution {
public:
    string ans;
    int count = 0;

    void backtracking(string &cur, int n, int k, int lastIndex) {
        if(cur.size() == n) {
            count++;
            if(count == k) ans = cur;
            return; // 终止递归
        }

        for(int i = 0; i < 3; i++) {
            if(i == lastIndex) continue;   // 相邻字符不能相同
            if(!ans.empty()) return;       // 找到第 k 个就停止
            cur.push_back('a' + i);
            backtracking(cur, n, k, i);
            cur.pop_back();
        }
    }

    string getHappyString(int n, int k) {
        string cur;
        backtracking(cur, n, k, -1); // -1 表示没有上一个字符
        return ans.empty() ? "" : ans;
    }
};

解题思路：
采用回溯算法，但是并不是列举所有，而是使得每次尽可能去先将小的字符放入，这样可以使得当数目满足k的时候
就可以直接返回了。
