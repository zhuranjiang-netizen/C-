## **解题代码（C++）**

```cpp
class Solution {
public:
    vector<int> resultsArray(vector<int>& nums, int k) {
        vector<int>result;
        int n=nums.size();
        int cnt=1;
        for(int i=0;i<n;i++)
        {
        if(i>0&&nums[i]==nums[i-1]+1)
        {
            cnt++;
        }
        else
        cnt=1;
        if(i>=k-1)
        {
            if(cnt>=k)
            result.push_back(nums[i]);
            else
            result.push_back(-1);
        }
        }
        return result;
    }
};
解题思路：
注意这里的要求是后一个元素比前一个元素多1，直接用cnt记录连续的满足要求的数字数，
如果此时的数目大于k的化就是直接将nums[i]放入，否则就是-1，因为满足要求的长度k的序列是
有限的，所以在i大于等于k-1的时候进行记录。
