## **解题代码（C++）**

```cpp
class Solution {
public:
    int numSpecial(vector<vector<int>>& mat) {
        int m=mat.size();
        int n=mat[0].size();
        int result=0;
        vector<int>countrow(m,0);
        vector<int>countline(n,0);
        for(int i=0;i<m;i++)
        {
         for(int j=0;j<n;j++)
         {
            if(mat[i][j]==1)
            {
                countrow[i]++;
                countline[j]++;
            }
         }
        }
        for(int i=0;i<m;i++)
        {
            for(int j=0;j<n;j++)
            {
                if(mat[i][j]==1&&countrow[i]==1&&countline[j]==1)
                result++;
            }
        }
        return result;
    }
};
解题思路：
先统计每行每列数据为1的个数，然后再次遍历一遍mat，如果满足行列上1的个数是1且当前数值为1，
result++。
