## **解题代码（C++）**

```cpp
class Solution {
public:
    int countSubmatrices(vector<vector<int>>& grid, int k) {
        int m = grid.size();
        int n = grid[0].size();
        int count = 0;

        vector<vector<int>> sum(m + 1, vector<int>(n + 1, 0));

        for(int i = 1; i <= m; i++) {
            for(int j = 1; j <= n; j++) {
                sum[i][j] = sum[i-1][j] 
                          + sum[i][j-1] 
                          - sum[i-1][j-1] 
                          + grid[i-1][j-1];

                if(sum[i][j] <= k)
                    count++;
            }
        }

        return count;
    }
};
解题思路：
使用二维前缀和。
