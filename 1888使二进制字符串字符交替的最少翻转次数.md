# **minimum-number-of-flips-to-make-the-binary-string-alternating**
**🔗 题目链接：**  
**[sum-of-root-to-leaf-binary-numbers](https://leetcode.cn/problems/minimum-number-of-flips-to-make-the-binary-string-alternating/)**

## **解题代码（C++）**

```cpp
class Solution {
public:
    int minFlips(string s) {
        int n = s.size();
        string ss = s + s;

        string t1 = "", t2 = "";
        for(int i = 0; i < 2 * n; i++)
        {
            if(i % 2 == 0)
            {
                t1 += '0';
                t2 += '1';
            }
            else
            {
                t1 += '1';
                t2 += '0';
            }
        }

        int diff1 = 0;
        int diff2 = 0;
        int ans = n;

        int l = 0;

        for(int r = 0; r < 2 * n; r++)
        {
            if(ss[r] != t1[r]) diff1++;
            if(ss[r] != t2[r]) diff2++;

            if(r - l + 1 > n)
            {
                if(ss[l] != t1[l]) diff1--;
                if(ss[l] != t2[l]) diff2--;
                l++;
            }

            if(r - l + 1 == n)
            {
                ans = min(ans, min(diff1, diff2));
            }
        }

        return ans;
    }
};
解题思路：
将字符串扩充一倍，这样未翻转的字符串必定是其子串。最后的情况总共有两种，一种是1开头的，一种是0开头的。
此时用滑动窗口去判断长度位n的窗口与两种情况不同字符的次数然后取最小值就可以了。
