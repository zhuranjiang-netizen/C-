# **predict-the-winner**

**🔗 题目链接：**  
**[predict-the-winner](https://leetcode.cn/problems/predict-the-winner/)**

## **解题代码（C++）**

```cpp
class Solution {
public:
    bool predictTheWinner(vector<int>& nums) {
        int n = nums.size();
        vector<vector<int>> dp(n, vector<int>(n, 0));

        for (int i = n - 1; i >= 0; i--) {
            dp[i][i] = nums[i];   // 区间长度为 1 的 base case

            for (int j = i + 1; j < n; j++) {
                dp[i][j] = max(
                    nums[i] - dp[i + 1][j],
                    nums[j] - dp[i][j - 1]
                );
            }
        }

        return dp[0][n - 1] >= 0;
    }
};
解题思路：
本题是采用动态规划博弈的算法，如果单从获得分数上面考虑可以发现题目是很难入手的。
换一个思路，我们从当前玩家比另一个玩家在某区间高出的分数来考虑，就会觉得豁然开朗。
这里将玩家直接模糊掉，只是考虑当前玩家，也就是即将在这个区间内选择分数的玩家，因
为这样推到到最后dp[0][n-1]不就是我们想要得到的吗。另外的话，遍历顺序可以由递推
公式得出来。
